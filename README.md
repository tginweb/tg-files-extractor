# tg-files-extractor

Utility to extract files from text with specific delimiters.

## Installation

```bash
npm install -g tg-files-extractor
# or
yarn global add tg-files-extractor
```

## Usage

```bash
tg-extract-files input.txt
```

## Input format

```
/* ==== path/to/file.js ==== */
Content of the file
/* ==== another/file.txt ==== */
Another content
```
